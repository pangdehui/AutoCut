- **排版**：清晰、层级分明

## 技术栈

### 前端
- React 19 + TypeScript
- Tailwind CSS 4 + shadcn/ui
- wouter（路由）
- tRPC（类型安全 RPC）
- Sonner（Toast 通知）

### 后端
- Node.js + Express
- tRPC 服务器
- Drizzle ORM
- MySQL/TiDB 数据库

### 工具（规划中）
- FFmpeg（视频处理）
- Whisper API（语音转文字）
- 多模态 AI 模型（内容分析）

---

## 注意事项

1. **邮件服务**：当前为模拟实现，需要集成真实服务（SendGrid/Mailgun）
2. **视频存储**：当前为本地存储，后期可升级为云存储
3. **AI 模型**：需要集成具体的多模态 AI 模型 API
4. **FFmpeg**：需要在服务器安装并配置路径
5. **异步任务**：需要实现任务队列系统（Bull/RabbitMQ）
